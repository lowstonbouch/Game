Game Tutorial Info

Tutorial

Run

To move the character, you need to press the buttons on the keyboard A or D

Jump

To jump a character, press the space bar.

Hook

To activate and start the hook, click on the left mouse button.

About game

Intro:

You have to go through complex tests in which you can rely only on your dexterity. Try not to get hurt! Use all improvised sires to reach the end.

Plagiarism:

The game is based on the popular game TeeWorlds. This game was created solely for the educational purpose for "The Rolling Scope School"
